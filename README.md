# 🌊 个人状态动力学追踪系统 (State Dynamics Tracker)

这是一个基于**双时间尺度弛豫模型 (Dual-timescale Relaxation Model)** 开发的个人状态与情绪追踪系统。

传统的打卡软件往往要求用户输入绝对分数，且完全忽略了人类记录行为在时间上的非等距性（即“非均匀采样”问题）。本项目将人类的心理与生理状态建模为一个具有内部阻尼和动态基线的物理演化系统。通过引入时间的连续流动，实现了极简输入与精准状态重构的完美结合。

## 🎯 项目目的

1. **消除绝对打卡负担**：人类对“绝对分数”的感知是模糊的，但对“相对当前基线的涨落”极其敏感。本项目仅需用户输入直觉感受的“相对增量”。
2. **解决非均匀采样问题**：生活的记录是离散且随机的。本系统引入绝对时间轴，状态的自然平复（衰减）由时间流逝本身驱动，而非记录次数。
3. **寻找局域平衡态**：情绪不会跌入绝对零点，而是向近期的“常态”（背景热浴）进行指数弛豫。

## 🧮 核心算法与物理图像

本系统由两个耦合的演化变量构成：
* **快变量 $S(t)$**：代表当下的绝对状态分数（Actual State）。
* **慢变量 $B(t)$**：代表近期的心理常态基线（Perception Baseline），即系统所处的局域准平衡态。

### 1. 状态的弛豫过程 (Time Evolution)
在没有任何外部事件触发的时间间隔 $\Delta t$ 内，状态 $S$ 会向基线 $B$ 发生指数弛豫（类似于牛顿冷却定律），同时基线 $B$ 也会被当前状态缓慢拉扯重塑。
设 $\tau_s$ 为状态弛豫时间（默认 2 天），$\tau_b$ 为基线演化时间（默认 14 天），其离散解析解为：

$$S(t + \Delta t) = B(t) + [S(t) - B(t)] \cdot e^{-\frac{\Delta t}{\tau_s}}$$

$$B(t + \Delta t) = B(t) + [S(t) - B(t)] \cdot (1 - e^{-\frac{\Delta t}{\tau_b}})$$

### 2. 事件触发的跃迁 (Event Transition)
当发生显著影响状态的事件时，用户输入一个模糊的相对感受冲量 $I \in \{-2, -1, 0, 1, 2\}$。此时，系统基线 $B$ 瞬间保持不变，而状态 $S$ 基于当前基线发生波函数坍缩（阶跃）：

$$S_{new} = B_{current} + I$$

这一修正确保了所有状态涨落都是以当前的“习惯常态”为参考系发生的，自然推导出人类状态的“软上限”与“阈值提高”效应。

## ✨ 主要特性

* **连续时间插值渲染**：在前端图表中，算法会自动在两次离散记录之间生成高频插值点，完美还原指数平滑演化的连续物理图像。
* **极简交互前端**：基于 Streamlit 构建，侧边栏一键点击即可完成状态跃迁记录。
* **数据本地持久化**：所有事件时间戳与输入将自动保存在同目录下的 `state_logs.csv` 中，安全且易于二次分析。
